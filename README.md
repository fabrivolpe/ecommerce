## General info
This project is a mockup ecommerce website that sells sports clothing. It allows the client to create a user, add products to the cart, and pay for said products.
	
## Technologies
Project is created with:
* HTML5
* CSS3
* JSES6
* PHP8
* SQL (MySQL)
* API (Stripe)
	
## Setup
To run this project:
1. Open https://onlysportsmyproject.000webhostapp.com/ in browser.
2. Browse through different pages of the website.
